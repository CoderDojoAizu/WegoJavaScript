1. [wego]クラス：基本動作を集めたクラス

1-1. おじぎをする

3. [wegoSensor]クラス：照度・距離・二次元コードスキャナー等センシング機能をもつクラス

3-1. 充電

  await commands.wego.bow();
}
await commands.wego.bye();

3-2. 体力

  await commands.wego.bow();
}
await commands.wego.bye();

3-3. 二次元コードリーダー

3-4. 周囲の明るさ

  await commands.wego.bow();
}
await commands.wego.bye();

3-5. 障害物までの距離

  await commands.wego.bow();

3-6. カメラの画像を解析する
